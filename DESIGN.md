# ChainSense · дизайн-система v2 «Meridian Light»

Светлая продуктовая система: лендинг продаёт реальными скриншотами платформы (паттерн Acumatica), жизнь даёт цветовая кодировка функциональных семей (паттерн Odoo), связность объясняет фирменный «живой контур».

## Шрифты (self-hosted, @fontsource, полная кириллица RU/AZ/EN)
- **Golos Text** 600/700/800 — display: заголовки, названия модулей.
- **Inter** 400/500/600 — текст, таблицы, контролы.
- **JetBrains Mono** 500/600 — приборный голос: коды SC-XX, eyebrow, значения KPI, числа таблиц, оси графиков.

## Цвет
- Primary **jade** `#0E7C63` (hover `#0B6450`) — единственный акцент действий.
- **Семь функциональных семей** (только навигация, иконки, легенды — не данные):
  jade `#149A7C` планирование · violet `#6A57D6` поставщики/документы · sky `#2E7BC4` логистика · amber `#CE861F` операции · coral `#CC4E2E` риск · plum `#B34983` AI · leaf `#54A25E` устойчивость. Формат FAMC: [tint, mid, deep].
- Данные в графиках — дисциплинированно: jade / slate `#5E7488` / brass `#B9893B` / signal `#BE4B32`.
- Нейтрали с pine-подтоном: canvas `#F4F6F2`, panel `#FFFFFF`, hairline `#E7EAE2`, ink `#101915`.

## Иконки
15 кастомных дуотон-глифов (`src/icons.jsx`, `ModIcon`), 24×24, 2–4 геометрические формы: deep+mid семьи + белые акценты на тонированном чипе. Используются в сайдбаре, шапке модуля и плитках лендинга.

## Кнопки — слоёная конструкция, ноль движения
Primary: градиент `#149478→#0C6B54`, бордер `#0A5D49`, inset-блик сверху, inset-тень снизу, ключевая тень jade. Hover — светлее+глубже тень; **active — inset-нажатие** (без transform). Ghost: белый, hairline, inset-блик; active — inset. Focus-visible кольцо на всём.

## Панели
Уровни поверхностей: canvas → panel (белая, hairline, тень 1px; hover — bloom тени) → sunken `#F7F8F4` (параметры, «Как пользоваться») → floating (hero-карточки, глубокие рассеянные тени) → dark instrument (тултипы графиков, реплики пользователя в чате). Сайдбар — светлая панель `#FBFCFA` с иконками-чипами; активный пункт: jade-заливка `#EAF3EE` + анимированный левый индикатор.

## Анимации (все отключаются prefers-reduced-motion)
- Сайдбар: индикатор активного пункта растёт/появляется (220ms cubic-bezier).
- KPI: pop значения при изменении (`.cs-vpop`, key по значению).
- Лендинг: scroll-reveal с каскадом; count-up цифр proof-полосы; поток по рёбрам контура (dash-offset); лёгкий float hero-чипов; hover плиток — bloom тени + масштаб иконки-чипа 1.07.
- Кнопки и строки таблиц — только цвет/тень, без смещений.
