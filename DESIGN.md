# ChainSense · Meridian design system

Редизайн-система, применённая ко всему продукту (лендинг + платформа). Идея: цепочка поставок как один инструментальный контур — «меридиан». Навигация — это и есть конвейер.

## Токены
| Роль | Значение |
|---|---|
| Jade (единственный акцент) | `#0E7C63` light / `#2EC29B` dark, hover `#0B6450` |
| Brass (внимание, дозированно) | `#B9893B` light / `#D9B36C` dark |
| Signal (только семантика риска) | `#BE4B32` |
| Slate (нейтральные ряды данных) | `#5E7488` / `#44586B` |
| Ink | `#141B17` текст · `#0F1613` сайдбар · `#0A100E` лендинг |
| Paper | фон `#F3F5F0` · surface `#FFFFFF` · sunken `#F7F8F4` · hairline `#E4E7E0` |

## Типографика
- **Display** — Schibsted Grotesk 700/800, tracking −0.02…−0.028em: заголовки, названия модулей.
- **Body** — Inter 400/500/600: текст, таблицы, элементы управления.
- **Instrument mono** — Spline Sans Mono 500/600: все коды (SC-XX), eyebrow-подписи, значения KPI, числа таблиц, оси графиков. Все цифры продукта говорят одним «приборным» голосом.
- Шрифты self-hosted через `@fontsource` — без внешних запросов.

## Сигнатура: меридиан-рельс и ромб-станция
- Сайдбар — вертикальный рельс с 15 станциями-ромбами (активная — заполненный jade-ромб с ореолом).
- На лендинге ромб + hairline открывают каждую секцию; узлы «живого контура» в hero — ромбы с кодами; диаграмма связности — метро-схема потоков SC-XX.
- Латунный ромб маркирует только узел риска — единственное отступление от jade.

## Поверхности
Три уровня: surface (белая карточка, hairline, тень только на hover) → sunken (`tone="sunken"`, панели параметров и блок «Методика») → dark instrument (тултипы, чат-реплика пользователя). Радиусы 9–12 px; тени рассеянные, без глянца.

## Состояния и микро-движение
Hover карточек — мягкий подъём тени; primary-кнопка — подъём + jade-свечение; слайдеры — залитый трек и масштаб бегунка; строки таблиц — тёплая подсветка; entrance `csUp`, дорисовка маршрутов `csDraw`, staggered `csFade`. Видимый `:focus-visible` на всех контролах; `prefers-reduced-motion` полностью отключает анимации.

## Графики
Единый голос через глобальные overrides (index.css): mono-оси `#8B948E`, сетка `#EEF0EA`, тёмный тултип `TIP` (App.jsx). Серии — только jade / slate / brass / signal.
